Predicting an animal

Launch How to use

Welcome to the animal prediction application {{ newani }}

This animal app predicts an animals class based on the information given,stores prediction in an api, and provides latest animal news after registering

  • It was built using a logistic regression machine learning model
  • It classifies prediction into seven classes:
  • Mammal,Bird,Reptile,Fish, Amphibian,Bug, Invetebrate
  • It was trained on the Zoo Classification Dataset on Kaggle

How to use

Follow the following steps to get started


1. Launch app

Click on launch app to get started


2. Picture an animal in your mind

Think of a beautiful animal


3. Fill in the form

Flll in a form containing some random Yes/No questions about animals generally


4. Get a predicted result

The ML model predicts the class of the animal you have thought about, using the infomation submitted in the form.


5. Fill in a feedback form after prediction

Select if it correctly predicted the animal class of the animal you were thinking. Also fill in the name of the specific animal you were thinking of.


6. Click on Add to API buttton

Add your animal feedback and ML prediction to an API to improve the animal datasets worldwide


7. View Api page

You now see your animal info added to the list of apis predicted by the animalpre app, you can also see other animals predicted in the api


8. Signup or Login

Register or login to access the animalpre news app that let you search for your favorite animals and read animal news.Also continue adding to Animal api
